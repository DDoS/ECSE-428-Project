# Gherkin Acceptance Test for Sorting Question Search Result

Feature: Sorting Questions Search Results
    Given I am an anonymous user or a registered / logged-in user
    And I want to view the searched questions in some order
    I should be able to sort the questions by date

Scenario: Obtain question search results in ascending order by date
	Given I open the url "http://mayhem-ecse428.rhcloud.com/questions/search"
	And the title is "Search Questions - Mayhem"
	When I set "keyword" to the input field "#search"
	And I select the option with the name "date_asc" for element "sort"
	And I click on the button ".btn.btn-search"
	Then I expect that the title is "Question Search Result - Mayhem"
	And I expect that element "#view_questions_date_asc" becomes visible

Scenario: Obtain question search results in descending order by date
	Given I open the url "http://mayhem-ecse428.rhcloud.com/questions/search"
	And the title is "Search Questions - Mayhem"
	When I set "keyword" to the input field "#search"
	And I select the option with the name "date_des" for element "sort"
	And I click on the button ".btn.btn-search"
	Then I expect that the title is "Question Search Result - Mayhem"
	And I expect that element "#view_questions_date_des" becomes visible

Scenario: Obtain question Search Results failed due to empty keywords
	Given I open the url "http://mayhem-ecse428.rhcloud.com/questions/search"
	And the title is "Search Questions - Mayhem"
	When I select the option with the name "date_des" for element "sort"
	And I click on the button ".btn.btn-search"
	Then I expect that the title is "Question Search Result - Mayhem"
	And I expect that element "#empty" becomes visible
	And I expect that element ".alert.alert-noKeyword" becomes visible

Scenario: Obtain question Search Results failed due to no questions
	Given I open the url "http://mayhem-ecse428.rhcloud.com/questions/search"
	And the title is "Search Questions - Mayhem"
	When I set "invalidKeyword" to the input field "#search"
	And I select the option with the name "date_des" for element "sort"
	And I click on the button ".btn.btn-search"
	Then I expect that the title is "Question Search Result - Mayhem"
	And I expect that element "#empty" becomes visible
	And I expect that element ".alert.alert-noMatch" becomes visible
